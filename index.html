Pagina 66 y 104

Esta pagina web tienen tanto el HTML como el CSS provistos por el libro y como se puede observar en la imagen, consiste en un formulario donde se registran inventarios del negocio "Ediciones Fares". Se puede asignar un codigo, precio y fecha a cualquier objeto que subamos por medio del boton "Seleccionar Archivo". La decoración fue dada por el libro.

Pagina 72

Esta pagina web tenia solo el HTML provisto por el libro pero con algunos errores, por lo que tuve que agregarle el CSS y el funcionamiento que tienen es el de registrar los datos para conseguir libros de las "Ediciones Fares" con casillas para seleccionar el curso y un boton para enviar datos.

Pagina 222

Este formulario originalmente usaba PHP y W3CSS como bases pero como no estamos trabajando con eso tuve que alteral el codigo y usar solo etiquetas HTML y CSS, el diseño aun no quedó como se muestra en el libro pero quizas haga cambios mas adelante.

Pagina 255

Esta pagina web consiste en una galería de imagenes y el libro solo dejo una imagen como instrucción a seguir, por lo que tanto el HTML como el CSS lo tuve que hacer, ademas las imagenes tienen un hover y hypervinculo especiales para tener una "vista previa". Como el resto de codigos decidí utilizar libros de la "Ediciones Fares" como base.
